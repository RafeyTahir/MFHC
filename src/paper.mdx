## Abstract

This work proposes a **hierarchical model-free control framework** for generating **stable limit cycles** in underactuated mechanical systems. The framework consists of a high-level controller that generates reference trajectories while ensuring stable internal dynamics, and a low-level **model-free controller** that tracks these references on the unactuated coordinates.

The proposed approach achieves accurate tracking without requiring explicit knowledge of the system dynamics, while retaining robustness and formal stability guarantees. The framework is validated on multiple benchmark systems, including the **Rotary Inverted Pendulum (RIP)** and **Vertical Take-Off and Landing (VTOL)** aircraft model through both simulations and real-time hardware experiments, and on a **leg–foot system on deformable ground** in simulation.

</HighlightedSection>

## Framework Overview

The control architecture is organized hierarchically.  
The **high-level controller** generates reference trajectories that induce a desired periodic motion, while the **low-level controller** ensures robust tracking of these trajectories despite uncertainty and underactuation.

## Simulation Studies

The framework is further evaluated through simulation on:
- Rotary Inverted Pendulum (RIP)
- Vertical Take-Off and Landing (VTOL) aircraft model
- Leg–foot system on deformable ground

All simulations confirm excellent tracking performance, robustness to uncertainty, and preservation of closed-loop stability.

---

## Software and Hardware Implementation

- **Simulation:** MATLAB / Simulink 2023b  
- **Hardware:** QNET 2.0 Rotary Pendulum with NI LabVIEW Suite 2021  

This repository provides all necessary code and configuration files to reproduce both simulation and experimental results.

---

## Safety Notes

When operating real systems, special care must be taken:

- Limit motion range, torque, and speed using software and hardware constraints.
- Maintain a safe distance from the robot during operation.
- Ensure access to an emergency stop at all times.
- Handle LiPo batteries with appropriate precautions.
